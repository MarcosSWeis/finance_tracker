## Instrucciones

### Terminal one

- 1. cd backEnd
- 2. npm i mysql2 -G
- 3. npm i --force
- 3. npm run createDB (este comando debe ejecutarlo la primera vez que crear e inicializar en su pc la base de datos)
- 4. npm start

### Terminal two

- 1.  cd frontEnd
- 2.  npm i --force
